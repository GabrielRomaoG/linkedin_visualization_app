Olá pessoal!! Vim apresentar um novo projeto que vou iniciar. A ideia base é fazer web scrapping de um perfil no Linkedin e apresentar essas informações num Dashboard. Os dados que eu vou extrair serão: ◾Informações sobre as conexões ◾Estatísticas das publicações (reações, comentários e reposts) ◾SSI ◾Visualizações no perfil ◾Ocorrências em pesquisa ◾Followers Usarei primariamente o Python para fazer a extração, tratamento e CRUD no Banco de dados. O objetivo é que a extração seja executada diariamente, então vou tentar automatizar essa parte utilizando algum solução na nuvem, como a AWS.
